# Quick Start

## English

### Install

```bash
git clone git@github.com:zhangjianle0836-netizen/ai-element-locator.git
cd ai-element-locator
```

1. Open `chrome://extensions/`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select this project folder

### Verify

1. Click the extension icon
2. You should see the `AI Element Locator` popup
3. Click `开始选择`
4. Move the mouse over any element on a normal website
5. The hovered element should be highlighted
6. Click the element to open the result panel

### Tips

- Use `Alt + Click` to capture the parent element
- Use `ESC` to exit selection mode
- Prefer clicking the real business element, not an inner icon or decorative span

## 中文

### 安装

```bash
git clone git@github.com:zhangjianle0836-netizen/ai-element-locator.git
cd ai-element-locator
```

1. 打开 `chrome://extensions/`
2. 开启 `开发者模式`
3. 点击 `加载已解压的扩展程序`
4. 选择当前项目目录

### 验证

1. 点击插件图标
2. 你应该能看到 `AI Element Locator` 弹窗
3. 点击 `开始选择`
4. 在普通网页中把鼠标移动到任意元素上
5. 当前悬停元素应被高亮
6. 点击元素后应出现结果面板

### 使用建议

- 按住 `Alt` 再点击，可以直接选择父级元素
- 按 `ESC` 可以快速退出选择模式
- 尽量点击业务元素本体，不要优先点击内部图标或装饰性文字
