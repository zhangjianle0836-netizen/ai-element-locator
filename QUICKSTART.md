# 🚀 快速开始指南

## 📦 第一步：生成图标（必做）

### 方法一：使用浏览器（推荐）

1. 在浏览器中打开 `generate-icons.html`
   ```bash
   # macOS
   open generate-icons.html

   # 或者直接双击文件
   ```

2. 点击页面上的"下载全部"按钮

3. 将下载的图标移到插件目录
   ```bash
   mv ~/Downloads/icon16.png ~/Downloads/icon48.png ~/Downloads/icon128.png .
   ```

### 方法二：手动创建（备选）

如果你熟悉图像处理，可以自己创建图标：
- 尺寸：16x16, 48x48, 128x128 像素
- 格式：PNG
- 文件名：icon16.png, icon48.png, icon128.png
- 建议设计：靶心/瞄准镜图标，紫色渐变背景

---

## 🔧 第二步：加载插件

### Chrome 浏览器

1. 打开 Chrome

2. 访问 `chrome://extensions/`

3. 开启右上角的"开发者模式"

4. 点击"加载已解压的扩展程序"

5. 选择 `element-locator` 文件夹

6. 成功！插件图标出现在工具栏

### Edge 浏览器

1. 打开 Edge

2. 访问 `edge://extensions/`

3. 后续步骤同 Chrome

---

## ✅ 第三步：验证安装

1. 点击插件图标

2. 应该看到一个紫色渐变的弹窗

3. 标题显示"🎯 元素定位器"

4. 有一个"开始选择"按钮

5. 如果看到这些，说明安装成功！

---

## 🎯 第四步：开始使用

### 测试一下

1. 打开任意网页（比如 https://www.baidu.com）

2. 点击插件图标

3. 点击"开始选择"按钮

4. 页面顶部显示提示："元素定位器已激活"

5. 鼠标悬停在任意元素上，会看到蓝色高亮

6. 点击元素

7. 显示"✓ 已复制到剪贴板"

8. 在任何文本编辑器中粘贴（Cmd/Ctrl + V）

9. 应该看到元素的详细信息！

### 与 Claude 配合

1. 打开 Claude Code

2. 使用插件复制元素信息

3. 粘贴给 Claude

4. 告诉 Claude 你想做什么

示例提示词：
```
我在页面上发现一个问题：

【元素定位信息】
（粘贴插件输出的内容）

【问题描述】
按钮颜色太浅，看不清楚

【期望效果】
加深按钮颜色，提高可读性
```

## ❓ 遇到问题？

### 图标无法下载

- 检查浏览器是否允许下载
- 尝试单独下载每个图标
- 使用备选方法手动创建

### 插件无法加载

- 确认图标文件都在目录中
- 检查 manifest.json 格式是否正确
- 查看 Chrome 扩展页面是否有错误提示

### 点击元素没有反应

- 刷新页面后重试
- 检查浏览器控制台是否有错误
- 某些网站可能限制插件运行

### 复制失败

- 检查浏览器剪贴板权限
- 尝试使用 http/https 协议的网站（非 file://）
